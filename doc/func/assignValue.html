asignacion de valores

el engine redis tiene ciertos conflictos con las comillas simples, por lo que si se asigna valores con las comillas este de manera

%{extends 'layout'}

lanzara una excepcion de compilacion.
solo se permite las comillas dobles en todas las funciones.

esta es la manera correcta.

%{extends "layout"}

incluso para imprimir datos puedes asignar valores

{! $age = 15 !} {! $name = "juan" !}

la concatenacion puedes usarla igual como lo hacia en php.

{! "mi nombre es: " . $name . " y mi edad es: " . 20 !}

toda las funciones de php estan disponibles.
