incluir o requerir archivos

el engine redis permite tambien utilizar la funcion de incluir o requerir un archivo utilizando las mismas palabras claves.

  1. include
  2. include_once
  3. require
  4. require_once

ejemplo

%{include "user/auth.php"} o %{require "user/login.redis.php"}
nota:

no es necesario especificar toda la ruta de donde se encuentra el archivo.

ya que los include y lo require se encuentra enlazados desde la ruta principal que haz especificado.
