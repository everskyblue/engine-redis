extender archivos

al utilizar el template con la extension .redis.php puedes extenderlo desde un archivo principal
donde se ejecute las salidas de bloque.

%{extends "layout.app"}

puedes llamar a un archivo utilizando el punto [.] o con el slash [/]

no es necesario especificar la extension ya solo comprobara un archivo con extension .php o .redis.php en caso de que no existe mandara un error.
