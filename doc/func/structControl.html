estructura de control

el engine redis cubre funcionalidades basicas de php como son las estructuras de control.


foreach

vista en la secci√≥n de empezando donde explica un poco la sintaxis.
el manejo de un foreach es igual que la de php con excepcion de ciertas cosas. donde evitas escribir muchos bloque de codigo php incrustado en un html aqui un ejemplo.


sintaxis de php

<?php
foreach(array(1, 2, 3) as $number):
?>
 <p><?php echo $number; ?></p>
<?php
endforeach;
?>

sintaxis de redis

%{foreach array(1, 2, 3) as $number}
  <p>{! $number !}</p>
%{endforeach}
