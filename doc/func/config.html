para unicizalizar la vista que requiera necesita pasar varios parametros a la clase View

<?php
use View\View;

$view = new View('directorio/vista', [
'compile' => 'storage/cache/',
'cache' => true
]);

el primer parametro a recibir es el directorio donde se cargara la vista, el segundo un array donde se le indica donde se guardara la cache del archivo compilado.
la segunda opcion indica verdadero guardara la cache esta opcion esta activada por default, si no quiere que se guarde el codigo compilado sera evaludado por la funcion eval.


generando vista

una vez especificado los parametros, incializa la vista.

$view->render('index')->create();

si su vista se encuentra en un siguiente directorio basta con hacerlo de esta manera

$view->render('user.signin')->create();

tambien puede asignar valores a la vista para ser utilizadas como variables.

$view->render('user.signin')->assign(['name' => 'juan'])->create();
